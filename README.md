# sugar-kingdom-cake-magic

「シュガー・キングダム 〜わたしだけの魔法ケーキ〜」は、感情診断とケーキ生成を組み合わせた女子向けWebアプリです。8つの質問に答えることで、あなたの心の状態に応じた16種類の魔法ケーキの中から最適なものを生成します。作成したケーキは図鑑に保存でき、SNSでシェアすることも可能。スマホ最適化された美しいUIで、空に浮かぶスイーツ王国「パティスフィア」の世界観を楽しめます。

## 技術スタック
- HTML5 / CSS3 / Vanilla JS
- Mobile-first responsive design
- Hosted on GitHub Pages
